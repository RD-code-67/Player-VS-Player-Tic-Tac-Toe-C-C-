# 井字棋 C++ 專案 (P v P / P v AI / AI v AI 資料獲取)

高中自主學習成果，用 C++ 寫的井字棋。

## 目前進度
- **人對人 (Player vs Player)**：基本盤面、座標轉換、勝負判定邏輯。
- **人對 AI (Player vs AI)**：跟電腦對弈的決策。
- **AI 對 AI (Data Acquisition)**：
- 原本：跑隨機對戰刷數據，把盤面狀態轉成 9 碼字串自動存進 DATA2.txt。
- 已更新：具有資料的機器人互下 並增加隨機性 以及增加分數比重。

## 開發環境
- 語言：C++
- 工具：Code::Blocks
